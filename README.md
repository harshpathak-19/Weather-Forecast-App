🌤️ Weather Forecast Web App

A responsive web application built with React.js, Tailwind CSS, Axios, and Chart.js, that provides real-time weather updates using the **OpenWeather API**.

 Features
- Search weather by city name 🌎  
- Real-time temperature, humidity, and wind speed  
- Interactive temperature trend graph 📊  
- Dark/Light mode toggle 🌙  
- Fully responsive design for all devices 📱  

 Tech Stack
React.js · Tailwind CSS · Axios · Chart.js · OpenWeather API
